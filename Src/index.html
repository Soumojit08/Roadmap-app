HTML Basics

  • Learn basic tags: "div, p, a, h1-h6" etc.
  • Semantic HTML: Use tags like header, footer, article, section for better accessibility and SEO.
  • Forms: Handle form inputs using input, textarea, select, and submit forms using button and form attributes like action and method
  • Tables: Create accessible tables using table, thead, tbody, tr, and td.

CSS Basics

  • Box Model: Understand content, padding, border, and margin
  • Positioning: Learn how to position elements using static, relative, absolute, and fixed.
  • Flexbox: Learn how to align and distribute space within a container. Use justify-content, align-items, flex-direction, and flex-wrap for responsive layouts.
  • CSS Grid: Create 2D layouts using grid containers and grid items. Work with grid properties: grid-template-columns, grid-template-rows, grid-gap.
  • CSS Transitions and Animations: Use transition for smooth changes to properties.Define keyframes using @keyframes for custom animations.
  • Preprocessors: Use SASS or LESS to write cleaner and modular CSS with variables and nesting.
  • Responsive design : Media query
